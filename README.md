# serverless-restful-dynamodb-server

Serverless RESTful server with DynamoDB

## Installation

Use the package manager [npm](https://www.npmjs.com/get-npm) to install serverless-restful-dynamodb-server dependencies.

```bash
npm install
```

## Usage

#### SERVERLESS OFFLINE

```bash
npm start
```
